# Existing Tools

Survey of the analysis packages, pipelines, and services already in use across the heliospheric radio community.
The goal of this page is to map the current landscape so the workshop can identify gaps and avoid re-inventing what already works.

This will eventually cover, for each tool:

- Maintainers and home repository
- Instruments supported
- Data products handled (raw, calibrated, browse, etc.)
- Language / runtime
- Licensing
- Strengths and known limitations

Contributions welcome.
Open a PR with an entry against your favourite package.
